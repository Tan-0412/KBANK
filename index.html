🔗 เชื่อมต่อ Google Sheets

ตั้งค่า Apps Script เพื่อซิงค์ข้อมูลกับ Google Sheet ของคุณครับ

1เปิด Google Sheet → Extensions → Apps Script
2วางโค้ดด้านล่าง จากนั้น แก้ SHEET_ID ให้ตรงกับ Sheet ของคุณ
ดู ID ได้จาก URL ของ Sheet: .../spreadsheets/d/[SHEET_ID]/edit
3กด Save แล้วไปที่ Deploy → New deployment → Web App
Execute as: Me  ·  Who can access: Anyone
4คัดลอก Web App URL มาวางด้านล่าง
